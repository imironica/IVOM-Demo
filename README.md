# TACAI-Demo
TACAI course demos

Instalation guideline

Install Python 3.5 from https://www.python.org/

Install the following Python packages (using pip or .whl packages from http://www.lfd.uci.edu/~gohlke/pythonlibs/):
       - numpy
       - cv2 (opencv for python)
       - scipy
       - pandas
       - mahotas
       - skimage
